- How is tableau extracted data different than a tableau live connection?

  - With extracted data you get a copy of the data from the data source. The data can be used anywhere without a connection. However as the name suggests, with a live connection, a connection is required to be established with the data source. Also, any changes in the source are reflected in the Tableau plots as well.

- What is the difference between these two tableau workbook file extensions `.twb` and `.twbx`?

  - With the .twb file no data is included with the tableau workbook. The user would need permission to access the source data. However with the `.twbx` file , an extract of the data is included.

- What is the difference between filters and parameters?

  - Filters are used to filter/segregate the data in the data source. You can add filters using the "Add" option on the Data Source page. Parameters/calculated parameters are like variables added to sheets / dashboards help to manipulate measures and dimensions in a sheet.
