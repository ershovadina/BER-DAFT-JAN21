- How are the two data sources that you imported, linked to each other?

If you look closely at the two files, in the spatial file there is a column called `GSS` code which represents a code for a district. And in the carbon emissions excel file, there is a column called code which also represents the district codes. Hence, similar to a join operation in SQL where we combine information from more than one table into a single table, we can do the same in Tableau as well. We will look at this operation in the later lesson.

Also, to view the file in a better structure in Tableau, you can use the data interpreter option. We will also talk about it in the next lesson.
